# Arqueólogo Git

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Arqueólogo Git para OpenClaw.

## Qué soluciona
Analiza el historial de git para extraer la justificación del código y generar documentación histórica.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque primero en seguridad
- Soporte para rollback

## Uso
/git-archaeologist

## Ejemplo
```
Analyze code for research issues.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
